# AI 完整产业链

1) 总体分层（概览）
- 能源/电力 → 算力（芯片、服务器、DC/边缘）→ 基础模型与算法 → 数据（采集/标注/合成/交易）→ 平台/中间件（MLOps、训练/推理）→ 行业应用（垂直化解决方案）→ 终端设备与传感 → 网络/连接 → 运维/能效 → 安全/合规 → 服务生态（SI、SaaS、咨询）→ 支撑产业（EDA、封测、材料）→ 创新前沿（量子、光算、神经拟态）

2) 分层细项、可投标的子赛道与关键投资要点
- 能源 / AI 专用电力
    - 可投对象：高密度数据中心用配电、专用变电+本地储能、液冷供电解决方案、能效优化软件
    - 投资要点：高CAPEX、长期回收、与数据中心客户绑定、政策与补贴敏感、ESG指标重要
    - KPI：PUE、能耗/算力、合同期、可再生比例

- 算力层（芯片与硬件）
    - 芯片设计：GPU/TPU/ASIC、神经网络加速器、IP核、FPGA定制
    - 制造/封测/代工、加速卡、服务器整机、加速器生态
    - 投资要点：技术壁垒高、长期订单可见性、客户集中、受制于晶圆供给与制程节点
    - KPI：算力/瓦、单片毛利、设计-量产周期、客户绑定度

- 数据中心与边缘算力
    - 集中DC、超大规模云、区域数据中心、边缘微数据中心、机柜托管
    - 投资要点：地产与电力占比高、长期合同（带宽/托管）、需评估占用率与扩容节奏
    - KPI：收入密度（$/U 或 $/MW）、利用率、合同年限、租户集中度

- 基础模型与算法（大模型生态）
    - 基础/通用模型、模型库、预训练与微调服务、模型压缩与蒸馏工具
    - 投资要点：IP 与数据壁垒、开源竞争、推理成本、推理优化为赢利点
    - KPI：模型性能（FLOPs vs 精度）、推理成本/请求、授权/商业化率

- 数据层（采集、标注、合成、隐私）
    - 数据采集平台、众包标注、合成数据（合成图像/语音）、数据交易所、隐私计算/联邦学习
    - 投资要点：数据质量与合规是核心、可复用数据资产可形成订阅收入
    - KPI：数据标注单价、复用率、合规风险、客户粘性

- 平台与中间件（训练/推理/MLOps）
    - 云端训练平台、分布式训练框架、模型管理、持续集成/部署、监控/可解释性工具
    - 投资要点：SaaS化、企业切换成本、扩展性、与主流云/算力绑定
    - KPI：ARR、粘性（DAU/MAU）、客单价、毛利率

- 行业应用（垂直SaaS/解决方案）
    - 金融风控、医疗影像、制药发现、智能制造、自动驾驶、安防、零售/推荐、内容生成/创作工具、企业搜索/知识库
    - 投资要点：垂直化场景能带来高毛利与高粘性，但需行业合规与定制开发成本
    - KPI：LTV/CAC、行业准入门槛、合规成本、单个客户营收

- 终端与传感层（IoT、摄像头、机器人）
    - 智能摄像、激光雷达、传感器套件、智能机器人、边缘设备
    - 投资要点：硬件+软件协同、量产与供应链管理、售后与服务收入
    - KPI：硬件毛利、单位出货成本、订阅/服务转换率

- 网络与连接（5G/光纤/边缘网关）
    - 专网、低延迟连接、边缘CDN、SLA合同
    - 投资要点：与运营商合作、长周期合同、地理覆盖策略
    - KPI：带宽收入、SLA违约率、覆盖率

- 运维与能效（散热、液冷、UPS）
    - 机房冷却、液冷一体机、能源管理软件、运维外包
    - 投资要点：提高能效可直接降低成本，差异化服务可收费
    - KPI：PUE改进百分比、运维成本占比、可靠性（MTBF）

- 安全、合规与治理
    - 对抗鲁棒性、模型审计、隐私合规工具、AI治理平台、审计日志
    - 投资要点：法规驱动需求，企业合规支出刚性强
    - KPI：合规通过率、潜在法律罚金暴露、收入来自合规模块占比

- 服务生态与系统集成
    - SI、行业咨询、定制开发、运营外包、托管服务
    - 投资要点：低技术壁垒但高客户黏性，适合并购整合
    - KPI：项目毛利率、复购率、客户集中度

- 支撑产业（EDA、材料、封测）
    - 半导体设计工具、特种材料、封装测试、硅光、热管理材料
    - 投资要点：与芯片周期同步，产业链位置决定回报稳定性
    - KPI：订单波动系数、技术门槛、客户粘性

- 创新前沿（量子、光算、神经形态）
    - 早期高风险高回报，长期替代潜力
    - 投资要点：非常高研发周期与不确定性，适合战略/长期基金或企业投资
    - KPI：里程碑达成、每次融资后的估值增量、技术可复制性

3) 横向商业化模式与退出路径
- 收费模式：SaaS订阅、按API/请求付费、按算力小时付费、授权许可、硬件销售+服务、数据订阅
- 资本回收与退出：并购（被云厂商/行业龙头收购）、IPO、二级市场回售、项目层SPV/资产证券化
- 投资阶段建议：基础设施/算力与能源——偏向成长/并购、基础模型与前沿技术——天使/风投/企业战略、行业SaaS/应用——早中期VC到成长PE、服务SI与托管——并购整合机会高

4) 风险与政策考量（投资人必须关注）
- 技术替代速度快、开源冲击、算力成本上升或供应链中断、数据与隐私监管、地缘政治（芯片与云限制）、高CAPEX导致资本耗尽
- 风险缓释：签订长期客户/电力合同、与云或超算中心建立战略合作、多区域布局、重资产与轻资产组合平衡

5) 关键尽调清单（简要）
- 技术可复制性与壁垒（专利/数据/客户）
- 单位经济（算力成本/推理成本、LTV/CAC）
- 合同与收入可见性（长期合同、SLA）
- 供应链与制造能力（芯片/硬件）
- 合规与数据治理风险
